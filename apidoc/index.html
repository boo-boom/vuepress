# .getLikedSorted

根据评论的点赞总数/置顶情况排序评论数据

  • 所属子系统:
  • 安全级别:Anonym
  • 接口分组:houseBsCntWrdApi
  • 接口状态:OPEN
  • 接口负责人:testname
  • 项目负责人:whooo

请求参数：

{
    "mid": 1,                                 // 必选 / number : 被评论的实体id
    "lastCntId": 1,                          // 可选 / number : 起始值传入0,后续每次传入获取到最后一条主评论的评论ID
    "pageOffser": "{offset: 0, limit: 20}"    // 必选 / string : 分页参数
}

响应参数：

{
    "value": [                                      // 评论组游标
        {
            "entityId": 1,                          // 被评论的实体id
            "cursor": 1,                            // 游标
            "commentList": [                        // 组内评论列表
                {
                    "id": 0,                        // 评论id
                    "userBrief": {
                        "id": 123,                  // 用户的id
                        "name": "qqqq",             // 用户的昵称
                        "avatar": "",               // 用户的头像
                        "signature": "",            // 签名
                        "mobile": "135********",    // 用户手机号
                        "userType": "COMMON"        // 用户类型 COMMON 普通用户, OFFICIAL 官方账号, OFFICIAL_SIMULATION_ACCOUNT 运营小号, OFFICIAL_UGC 官方UGC
                    },
                    "state": "CREATED",             // 评论状态 CREATED 已创建, DELETE 删除, SHIELD 屏蔽
                    "comment": "",                  // 评论内容
                    "repliedDetail": {       // 被评论的详细信息

                    },
                    "commentateTime": 12334,        // 评论时间
                    "likedCount": 10,               // 获赞数
                    "commentTop": true,             // 是否置顶
                    "liked": false,                 // 是否已点赞
                }
            ]
        }
    ]
}
